
### Usage
```
usage: text_repo.py [-h] [-t TOKEN] path

Concatenate files from a local directory or GitHub repository, estimate token length,

positional arguments:
  path                  Local directory path or GitHub repository full name (e.g., 'owner/repo')

options:
  -h, --help            show this help message and exit
  -t TOKEN, --token TOKEN
                        GitHub Personal Access Token (required for GitHub repositories)
```
